Это инструкция.

Здравствуйте дорогие уважаемые читатели. Сегодня покажем как сделать вот так:

Анимация работает только в браузере, мобильный клиент не проигрывает анимацию. Но с мобильного клиента можно отправлять ранее отправленные граффити, в том числе и анимированные.

Шаги:

  • GIF -> aPNG
  • Получение токена ВК
  • Осознание айди собеседника
  • Вставить данные в раздел сайта

Получаем ссылку на aPNG.

Первым делом надо зайти на ezgif и сконвертировать Ваш gif файл в файл формата apng.

Затем скопируйте ссылку на получившийся результат(пкм -> копировать адрес картинки)

Проверьте скопированный результат: вставьте в новой вкладке ссылку. Браузер должен вернуть только картинку, а адрес должен заканчиваться на .png

Теперь Вы можете вставить ссылку в поле LINK (сохраните вкладку, она ещё понадобится)

Получаем токен

Токен лучше никогда никому не передавать. Но я (честно) его не украду.

Токен можно получить тут: vkhost.github.io

Для работы приложения надо предоставить доступ к сообщениям и документам

Нажмите получить, а затем разрешите приложению доступ к данным.

Откроется пустая вкладка. Скопируйте из адресной строки токен. Он расположен между access_token= и &expires in.

Вставьте токен в поле TOKEN.

Получаем айди собеседника

Зайдите в диалог и скопируйте из адресной строки цифры после sel=.

Если же Вы хотите отправить в беседу, то прибавьте к цифрам после sel 2 миллиарда.

Например, если айди беседы 125, то в поле PEER ID надо вставить 2000000125

RANDOM ID

В поле random id надо ввести случайное число. Оно нужно для метода апи, чтобы одно сообщение случайно не отправилось дважды

Profit.

Всё, теперь у нас есть все необходимые данные для отправки анимированного граффити.

Вставьте в поля соответствующие им значения и нажмите кнопку SUBMIT.

НАДО ПОДОЖДАТЬ НЕСКОЛЬКО СЕКУНД

Если всё хорошо, то выведется MESSAGE_ID: и тут много цифр.

Если какого-то параметра нет, то выведется -1

Если что-то пошло не так с картинкой .png, то выведется INVALID PIC

Если что-то не так с обращением к апи вк, то выведется класс ошибки.

Если что-то не работает, то хорошо бы написать: vk
